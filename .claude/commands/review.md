Проведи code review текущих изменений.

Шаги:
1. Запусти `git diff` и `git diff --staged` и изучи изменения
2. Проверь соответствие конвенциям из CLAUDE.md
3. Прочитай `docs/decisions.md` — свери с архитектурными решениями
4. Проверь правило языка: код, комментарии — на английском
5. Для firmware изменений:
   - Не обходит ли код `modeSet()` при записи в ACCMODE?
   - Сохраняется ли SELFREQ бит при смене режима?
   - Проверь модульность, отсутствие глобального состояния, корректность SPI операций
6. Для Python изменений:
   - Нет ли Qt-импортов в `core/`?
   - Обновляются ли `ctrl.set_mode()` и `plots.set_mode()` синхронно?
   - Проверь thread-safety SerialReader, корректность numpy-операций
7. Дай резюме на русском: что хорошо, что исправить (указывай файл:строка)

Будь конкретным. Не переписывай код без запроса — только предлагай.
