# ADE9000-PhaseMonitor

Real-time three-phase (delta) voltage monitoring system based on ADE9000 and Arduino Zero.

## Features

- Measure Uab, Ubc, Uca (line-to-line voltages)
- Detect voltage dips and imbalance
- Real-time visualization on PC
- Event capture and logging
- Designed for power supply startup analysis

## Hardware

- Arduino Zero
- EV-ADE9000SHIELDZ
- Modified voltage dividers for 400V delta

## Software

- Arduino firmware (C++)
- PC application (Python)
